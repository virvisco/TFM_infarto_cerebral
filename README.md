# Predicción de ocurrencia de infarto cerebral

Este repositorio contiene el código desarrollado para el Trabajo Fin de Máster del Máster en Bioestadística y Bioinformática (CEMP).

## Contenido
- `TFM_infarto_cerebral.Rmd`: archivo principal con el análisis completo
- Scripts en R para el análisis estadístico y la modelización predictiva
- Figuras y tablas generadas a partir del análisis

## Reproducibilidad
El análisis se ha realizado en R (RStudio) utilizando paquetes ampliamente empleados en bioestadística y análisis de datos clínicos.
